- **Show HN: Browser Harness――LLMにブラウザタスクの完全な自律実行を与えるOSSフレームワーク**
  フレームワーク制約を取り除き、LLMが事前学習知識に基づいてブラウザ操作を自律的に決定・実行できるOSSハーネスを公開。自己修正機能とツールの動的追加に対応する。
  → [Hacker News](https://news.ycombinator.com/item?id=47890841)
  <details><summary>📖 詳しく読む</summary>
  Browser Harnessは、LLMがブラウザタスクを実行する際に詳細なフレームワーク制約を課さず、モデルが事前学習から得た知識に基づき自律的に判断・完結できるよう設計されたOSSフレームワーク。実行中のエラーからの自己修正（self-healing）機能を内蔵し、新しいブラウザ操作ツールを実行時に動的追加できる。LLMエージェントのブラウザ自動化実用化に向けたアプローチとしてHacker Newsの開発者コミュニティで議論を集めた。
  </details>

- **Show HN: Agent Context――AIコーディングツールに参照プロジェクトの文脈を自動提供**
  Cursor・Claude Code・ChatGPTなどのAIコーディングツールに、開発者が指定した参照プロジェクトのコンテキストを自動的に提供するOSSツール。類似コードベースのスタイルやアーキテクチャをAIに認識させ、一貫性のある提案を引き出す。
  → [Hacker News](https://news.ycombinator.com/item?id=47919538)
  <details><summary>📖 詳しく読む</summary>
  Agent Contextは、AIコーディングアシスタント（Cursor、Claude Code、ChatGPT等）に対して参照したいプロジェクトのソースコードやドキュメントを文脈として自動提供するOSSツール。実プロジェクトのコーディングスタイル・設計パターン・アーキテクチャをAIに認識させることで、生成コードの一貫性を高める。コンテキストウィンドウの管理を抽象化する設計がHNコミュニティで評価された。
  </details>

- **Show HN: 無料のllms.txtジェネレーター――WebサイトURLから自動生成**
  WebサイトのURLを入力するだけで、LLM向けコンテキストファイル「llms.txt」を自動生成する無料ツール。AIエージェントがWebコンテンツの構造を把握するための標準形式への対応を容易にする。
  → [Hacker News](https://news.ycombinator.com/item?id=47901865)
  <details><summary>📖 詳しく読む</summary>
  llms.txtはLLMやAIエージェントがWebサイトのコンテンツ構造を把握するための標準形式ファイル。このツールはURLを入力すると自動的にllms.txtを生成し、手動作成の手間を省く。robots.txtの「AI版」とも言える位置づけで、AIエージェントによる適切なコンテンツ利用の促進を目的として普及が進んでいる。
  </details>

- **cPanel・WHMに認証バイパス脆弱性CVE-2026-41940（CVSS 9.8）――150万インスタンスが危険にさらされ野外悪用を確認**
  cPanelおよびWebHost Manager（WHM）の全対応バージョンに深刻な認証バイパス脆弱性CVE-2026-41940が発見され、CISA・各ベンダーが緊急パッチの適用を呼びかけている。攻撃者は2026年2月23日以前から悪用しており、150万以上のインスタンスが影響を受ける可能性がある。
  → [The Hacker News](https://thehackernews.com/2026/04/critical-cpanel-authentication.html)
  <details><summary>📖 詳しく読む</summary>
  CVE-2026-41940はCRLFインジェクションを悪用した認証バイパス脆弱性。攻撃者はログイン処理中に`whostmgrsession`クッキーの一部を省略しつつ、Basic Authorizationヘッダーに`\r\n`文字を注入することでセッションファイルに任意の属性（`user=root`等）を書き込める。cPanel 11.40以降の全バージョンに影響し、WP Squared（cPanelベースのWordPressホスティング管理パネル）も対象。緊急パッチが2026年4月28日にリリース済み。CISAは活発な悪用（2月23日以前から継続中）を確認し、政府機関へ即時パッチ適用を義務付けた。
  </details>

- **SAP公式npmパッケージ4本がサプライチェーン攻撃で侵害――CI/CD認証情報やブラウザパスワードを窃取**
  SAP Cloud Application Programming Model（CAP）で使用される公式npmパッケージ4本（@cap-js/sqlite等）に悪意のあるコードが混入し、開発者のGitHub・npmトークン、クラウドシークレット（AWS・Azure・GCP等）、ブラウザ保存パスワードなどが窃取された。
  → [The Hacker News](https://thehackernews.com/2026/04/sap-npm-packages-compromised-by-mini.html)
  <details><summary>📖 詳しく読む</summary>
  2026年4月29日に詳細が公開された「Mini Shai-Hulud」と命名されたサプライチェーン攻撃。侵害パッケージは@cap-js/sqlite v2.2.2、@cap-js/postgres v2.2.2、@cap-js/db-service v2.10.1、mbt v1.2.48の4本。preinstallスクリプトを利用してBunランタイムをGitHubからダウンロードし、難読化されたペイロードを実行する。盗取された認証情報はGitHubパブリックリポジトリ経由で外部へ送信される。TeamPCPグループとの関与が中程度の信頼度で示されており、Trivy・Checkmarx・Bitwardenへの過去攻撃との技術的類似点が確認されている。
  </details>
