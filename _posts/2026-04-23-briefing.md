- **MicrosoftがASP.NET Core CVE-2026-40372の帯域外パッチを緊急リリース、権限昇格が可能なCVSS 9.1の深刻な脆弱性**
  MicrosoftがASP.NET Coreに存在するCVSS 9.1の重大な脆弱性CVE-2026-40372を修正する帯域外パッチを緊急リリースした。暗号署名の不適切な検証によりネットワーク経由での権限昇格が可能となる。
  → [The Hacker News](https://thehackernews.com/2026/04/microsoft-patches-critical-aspnet-core.html)
  <details><summary>📖 詳しく読む</summary>
  CVE-2026-40372はASP.NET Coreにおける暗号署名の不適切な検証に起因する権限昇格脆弱性（CVSS: 9.1）。未認証の攻撃者がネットワーク経由でシステム権限を昇格させることが可能。定例パッチではなく帯域外（アウトオブバンド）での緊急リリースとなっており、深刻度の高さを示している。ASP.NET Core を使用したWebアプリケーション・APIサービスを運用している組織は即時のパッチ適用が推奨される。匿名の研究者によって発見・報告された。
  </details>

- **BRIDGE:BREAK：Lantronix・Silexのシリアル変換器に22件の脆弱性、2万台超のデバイスにリスク**
  産業用シリアル-IPコンバーター製品（Lantronix・Silex製）に22件の脆弱性「BRIDGE:BREAK」が発見された。インターネット上に公開されている約2万台以上のデバイスが乗っ取りとデータ改ざんのリスクにさらされている。
  → [The Hacker News](https://thehackernews.com/2026/04/22-bridgebreak-flaws-expose-20000.html)
  <details><summary>📖 詳しく読む</summary>
  BRIDGE:BREAKと命名された22件の脆弱性群は、産業ネットワークで広く使われるLantronixおよびSilexのシリアル-IPコンバーターに存在する。これらのデバイスは工場・病院・インフラ施設などで古いシリアル機器をネットワーク接続するために使われており、侵害された場合は産業制御システム（ICS）への攻撃経路となりうる。ShodanのデータによるとインターネットにEXPOSEDされている対象デバイスは2万台超。デバイスの乗っ取りやデータ改ざんが可能で、製造・医療・インフラ分野での影響が懸念される。
  </details>

  IPAが2026年4月22日に公開されたOracle Java（Oracle Java SE）の脆弱性対策に関する重要情報を公表した。悪用された場合、アプリケーションの異常終了などの影響が生じる可能性があるとして、更新プログラムの速やかな適用を呼びかけている。
