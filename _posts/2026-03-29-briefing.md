<details markdown="block">
<summary>🔒 セキュリティ（4件）</summary>

- **Microsoft 3月Patch Tuesday — 84件の脆弱性を修正、ゼロデイ2件含む**
  .NET のDoS脆弱性（CVE-2026-26127）と SQL Server の権限昇格脆弱性（CVE-2026-21262）が公開済みゼロデイとして修正された。緊急度「Critical」が8件。
  → [The Hacker News](https://thehackernews.com/2026/03/microsoft-patches-84-flaws-in-march.html)

- **GNU InetUtils telnetd に未修正の重大RCE脆弱性（CVE-2026-32746、CVSS 9.8）**
  未認証の遠隔攻撃者が root 権限で任意のコードを実行できる脆弱性。パッチは未提供のためtelnetd の無効化が推奨される。
  → [The Hacker News](https://thehackernews.com/2026/03/critical-telnetd-flaw-cve-2026-32746.html)

- **Google Chrome のゼロデイ2件が実際の攻撃に悪用 — CISA が KEV カタログへ追加**
  Skia と V8 に存在するゼロデイが野放しで悪用されており、CISAは3/27までの適用を連邦機関に義務付けた。一般ユーザーも速やかなアップデートが必要。
  → [The Hacker News](https://thehackernews.com/2026/03/google-fixes-two-chrome-zero-days.html)

- **IPA が 2026年3月の Microsoft 製品脆弱性対策情報を公開**
  3月11日付でIPAが月例のセキュリティ更新情報を公開。悪用された場合の影響（任意コード実行・PCの乗っ取り）が明記されており、早急な適用を呼びかけている。
  → [IPA](https://www.ipa.go.jp/security/security-alert/2025/0311-ms.html)

</details>
